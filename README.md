# C++ Tidbits
This repository contains some example code and problems I do in my free time or during lunch breaks, think of them more like puzzels or fun facts.  

## How to build
- Clone this repository
- Go to the root folder that contains `CMakeLists.txt` and run this command in CMD `cmake -B build`
- Your solution file will now be placed in the `build` folder  

## Topics
The Current topics that are covered in this repository are:  
- A Small function that beeps every 3 seconds using `Chrono`
- Example code that overloads the Post and Pre Increment operators
- A function that matches sub-string pattern using only pointer arithmetic
- A function that reverse string using only pointer arithmetic
